สรุปการแก้ไขระบบ Export Product Price (ICCAT_CODE Filter)

1. ไฟล์หน้าจอหลัก: export_data_product_price.php

  • เพิ่มระบบเลือกกลุ่มสินค้า: ดึงข้อมูล ICCAT_CODE ทั้งหมดจากฐานข้อมูล SQL Server มาแสดงเป็น Checkbox
  • ระบบเลือกทั้งหมด (Select All): เพิ่ม Checkbox "เลือกทั้งหมด" พร้อม JavaScript เพื่อให้เลือก/ไม่เลือกทุกรายการได้ในคลิกเดียว
  • ค่าเริ่มต้น: ตั้งค่าให้เลือกทุก Checkbox (Default Checked) เมื่อเปิดหน้าจอ
  • การกรองข้อมูล: ยกเว้นกลุ่มสินค้าที่ไม่ต้องการแสดงในรายการ ได้แก่:
    • TATA-001
    • RR.เสื้อผ้า
    • TATA-006

2. ไฟล์ประมวลผลหลังบ้าน: export_process/export_data_product_prices.php

  • รองรับเงื่อนไขใหม่: ปรับปรุงให้รับค่า iccat_code[] จากหน้าจอ
  • ปรับ Query: ใช้คำสั่ง IN (...) เมื่อมีการเลือก Checkbox เพื่อให้ได้ข้อมูลตามกลุ่มที่เลือกจริง
  • กรณีไม่ได้เลือก: หากไม่มีการเลือก Checkbox เลย ระบบจะดึงข้อมูลทั้งหมด (1=1)

3. การทำความสะอาดโค้ด (Cleanup): cond_file/query-product-price-main.php

  • ยกเลิกเงื่อนไข Hardcode: ลบเงื่อนไขเดิมที่จำกัดเฉพาะ 1SAC% - 9SAC% ออก
  • เปลี่ยนเป็น Dynamic: เพื่อให้ระบบรองรับการเลือกกลุ่มสินค้าได้อย่างอิสระผ่านหน้าจอ
